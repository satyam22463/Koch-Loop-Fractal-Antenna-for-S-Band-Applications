# Koch Loop Fractal Antenna — S-Band

A printed Koch Loop Fractal Antenna designed for S-band wireless applications (2.4–3.0 GHz), built on FR-4 substrate and simulated in CST Studio Suite.

---

## What It Does

Uses fractal geometry to pack a longer electrical path into a compact PCB footprint, achieving wideband S-band coverage without increasing the physical size of the antenna.

---

## Key Results

| Parameter | Value |
|---|---|
| Resonant Frequency | 2.93 GHz |
| Return Loss $S_{11}$ | −23.92 dB |
| Bandwidth (−10 dB) | 1.00 GHz (2.327 – 3.332 GHz) |
| Secondary Resonance | 6.7 – 7.2 GHz |

---

## Stack

- **Substrate:** FR-4 ($\varepsilon_r \approx 4.4$)
- **Feed:** 50 Ω microstrip + edge-launched SMA
- **Simulation:** CST Studio Suite
- **Validation:** VNA measurement
